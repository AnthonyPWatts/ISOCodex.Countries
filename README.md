# ISOCodex.Countries

`ISOCodex.Countries` is a planned .NET library for strongly typed country, country-code, subdivision, and jurisdiction metadata.

It is intended as a standards-aware domain foundation package for the broader ISOCodex family. It is not an official ISO product, is not endorsed by ISO, and is not a geopolitical authority.

Initial implementation work is described in [`01_FIRST_PASS_ISOCODEX_COUNTRIES.md`](01_FIRST_PASS_ISOCODEX_COUNTRIES.md).

## Current status

Bootstrap scaffold only. No package has been implemented yet.
