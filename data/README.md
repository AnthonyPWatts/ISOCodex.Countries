# Data

This folder is for source-aligned seed or generated data used by `ISOCodex.Countries`.

The library exists to keep country and subdivision handling predictable in .NET applications without hiding live lookups or provider-specific assumptions in runtime code. Data kept here should therefore be traceable, redistributable, and suitable for generating stable typed registries.

Data files must include provenance and version details. Do not add copied data unless redistribution rights are clear.
