# Data Sources

`ISOCodex.Countries` is intended to align conceptually with ISO 3166 country-code concepts, UN M49 numeric codes where applicable, and CLDR-style display-name thinking where appropriate.

This project is not an official ISO product and does not claim ISO endorsement.

The first implementation may use a small, safe seed dataset if redistribution or licensing questions make comprehensive source data unsuitable for immediate inclusion. Future work should improve source coverage transparently, with provenance and version details recorded alongside any added data.
